Leaflet Geojson bbox layer example

Example of using geojson bounding box leaflet layer to show point and polygon layers. Bbox layers only work when zoomed in enough. The point layer has example code how to respond to click events.

For comparison a vanilla leaflet geojson layer is provided. This layer uses the same endpoint as the point layer but shows all features at the same time with the expected performance hit.
